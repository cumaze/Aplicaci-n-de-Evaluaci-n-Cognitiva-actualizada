# **App Name**: App de Evaluación Cognitiva

## Core Features:

- Onboarding de Usuario: Recopila el nombre, el correo electrónico, la carrera y el nivel de experiencia del usuario para una evaluación personalizada.
- Generación Dinámica de Preguntas: Genera preguntas para cada competencia, adaptadas a la carrera y el nivel de experiencia del usuario utilizando una herramienta de IA generativa.
- Visualización de Preguntas y Respuestas: Muestra una pregunta a la vez y registra las respuestas del usuario.
- Evaluación de Competencias: Determina las competencias relevantes en función de la carrera seleccionada, informando la generación de preguntas por parte de la IA.
- Generación de Informes: Genera un informe que muestra las respuestas del usuario y la información básica.

## Style Guidelines:

- Color primario: Azul suave (#79A7D3) para crear un ambiente tranquilo y profesional, inspirado en conceptos de claridad y confiabilidad.
- Color de fondo: Azul muy claro (#F0F4F8), una versión desaturada del primario, para un fondo limpio.
- Color de acento: Morado apagado (#9973B2), un color análogo al primario, utilizado con moderación para resaltar acciones y elementos clave.
- Fuente del título: 'Space Grotesk' (sans-serif) para titulares y secciones de texto más cortas. Fuente del cuerpo: 'Inter' (sans-serif) para textos más largos.
- El diseño presentará un diseño de una sola columna para las interfaces de incorporación y preguntas, centrándose en la legibilidad y la facilidad de uso.
